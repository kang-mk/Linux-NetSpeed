# Linux-NetSpeed
```
wget -N --no-check-certificate "https://raw.githubusercontent.com/kang-mk/Linux-NetSpeed/master/tcp.sh"
chmod +x tcp.sh
./tcp.sh
```



脚本说明  
支持系统  
Centos 6+ / Debian 7+ / Ubuntu 14+  
BBR魔改版不支持Debian 8
